# Project Memory

## Core
Garak (جارك) — community marketplace for Dar Misr Al-Andalus, New Cairo, Egypt.
Primary teal #0D9488, accent coral #F97316, bg #F8FAFC. Outfit headings, IBM Plex Sans Arabic body.
Bilingual AR+EN with RTL support. Phase 1: responsive web, Phase 2: mobile app.
Lovable Cloud backend (auth, db, storage). Phone-to-email auth pattern (phone@garak.eg).
Free at launch, monetize later. Auto-confirm email enabled.
